Recursos para tutorialSsart/formatos (mismos del tutorial original)

1) creep.mp4 (recomendado 1280x720)
2) foto.png (poster recomendado 1280x720)
3) mp3b.png (recomendado 256x256)
4) ogg.png (recomendado 256x256)
5) scor.mp3 (recomendado 128kbps o superior)

Nota:
- No necesitas un tamano fijo universal.
- Ajusta dimensiones con CSS segun cada seccion.
- Mantener proporciones del video/poster mejora resultado visual.
