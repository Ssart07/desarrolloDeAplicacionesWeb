Sombras de <div> (cajas)

Ejemplo

Con HTML5 podemos crear de una manera sencilla tanto sombras para zonas del HTML (<div>, <article>, <footer>, <header>...), como para textos. Aunque cada uno de estos tipos de sombra tiene su propia etiqueta, la sintaxis sigue la misma logica.

Sombras internas

Para crear sombras internas unicamente tenemos que anadir el valor "inset" al final de la propiedad "box-shadow".


  • (6px), indica el numero de pixeles que se desviara la sombra horizontalmente.
  • (20px), especifica el numero de pixeles que se movera la sombra verticalmente.
  • (10px), es la suavidad que tendra la sombra (a menor valor mas duro sera el borde de la sombra).

creacion de 'halos' o 'resplandores'

Las sombras en CSS pueden utilizarse para generar efectos de halo o resplandor alrededor de elementos, destacandolos visualmente. Este recurso es util para llamar la atencion sobre componentes importantes o crear estilos mas llamativos.

Ejemplo

section{ box-shadow: 0px 0px 30px blue; }

Sombras en textos

En los textos, las sombras permiten mejorar la legibilidad y anadir un toque visual distintivo. Dependiendo de su configuracion, pueden hacer que el contenido resalte o darle un efecto mas suave y elegante.

multiples sombras en el mismo elemento

CSS permite aplicar varias sombras a un solo elemento, combinandolas para lograr efectos mas complejos. Esto facilita la creacion de disenos mas elaborados y personalizados sin necesidad de recursos adicionales.

Ejemplo

#tutorial_apartado_texto{ box-shadow: 0px 0px 20px red inset, 0px 0px 25px blue; }

La primera sombra (0px 0px 20px red inset) se posiciona directamente sobre el elemento, con un difuminado de 20 pixeles, color rojo y aplicada de forma interna. Por su parte, la segunda (0px 0px 25px blue) tambien se ubica sobre el elemento, con una extension de 25 pixeles y color azul; al no incluir la propiedad inset, se interpreta como una sombra externa.
