# Desarrollo de Aplicaciones Web

Tutorial de **HTML5 + CSS** desarrollado como material de estudio para el curso de Desarrollo de Aplicaciones Web.

## Contenido del repositorio

```
tutorial-html-css/
├── index.html   # Página principal con ejemplos de HTML5 semántico y formularios
└── styles.css   # Hoja de estilos con ejemplos de CSS (Flexbox, sombras, formularios)
```

## Temas cubiertos

- **HTML5**: Elementos semánticos (`header`, `nav`, `main`, `section`, `article`, `footer`), formularios con validación.
- **CSS**: Estilos generales, modelo de caja, Flexbox, transiciones, diseño responsivo.

## Cómo ver el proyecto

1. Abre la carpeta `tutorial-html-css/`.
2. Haz doble clic en `index.html` para abrirlo en el navegador, o usa una extensión como **Live Server** en VS Code.
