Lineas separadoras en HTML y CSS

Para dividir visualmente el contenido, se pueden usar lineas separadoras. En el caso de las horizontales, la forma mas sencilla es mediante la etiqueta < hr >, la cual no requiere cierre y genera una linea automatica en la pagina. Tambien es posible personalizar su apariencia utilizando estilos CSS. Para lineas verticales, generalmente se recurre al uso de bordes aplicados a elementos.

Tipos de bordes disponibles en CSS Los estilos de borde permiten modificar el aspecto de estas lineas. Entre los mas comunes estan:
none: elimina cualquier borde.
solid: con linea continua (valor predeterminado).
dotted:crea una linea formada por puntos.
dashed: genera una linea segmentada con guiones.
inset: crea un efecto de borde hacia adentro (hendidura).
outset: crea un efecto de borde hacia afuera (relieve).

Lineas separadoras verticales

A diferencia de las lineas horizontales, en HTML no existe una etiqueta especifica para crear lineas verticales. Para lograrlas, se utilizan los bordes de un elemento (como un < div>) mediante propiedades de CSS. Estas permiten definir una linea en cualquiera de los lados del contenedor.

Para crear lineas verticales, se deben definir uno de los cuatro bordes que tiene una caja (< div>), utilizando una (o varias) de estas propiedades:

    border-left: 1px inset orange;
    border-right: 1px dashed #3C5391;
    border-top: 1px solid red;
    border-bottom: 2px dotted green;
Si se desea una unica linea vertical, basta con aplicar una de estas propiedades al elemento.

Bordes simples en CSS

borde simple

Los bordes en CSS permiten delimitar visualmente elementos dentro de una pagina, ayudando a organizar mejor el contenido. A traves de sus propiedades, es posible ajustar caracteristicas como el grosor, el estilo, el color y el redondeo de las esquinas. Esto facilita crear disenos mas claros y atractivos.
