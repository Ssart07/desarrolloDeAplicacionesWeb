HTML (HyperText Markup Language)

HTML es el lenguaje estandar utilizado para crear y estructurar el contenido de las paginas web. Permite definir elementos como titulos, parrafos, imagenes, enlaces y listas mediante etiquetas. Su funcion principal es organizar la informacion de forma semantica para que los navegadores puedan interpretarla correctamente. Es la base sobre la que se construye cualquier sitio web.

Aspectos clave de HTML:

  • Estructura semantica:Organiza el contenido usando etiquetas con significado (como "header", "article", "footer").
  • Uso de etiquetas:Define elementos mediante etiquetas que indican su funcion dentro de la pagina.
  • Jerarquia de contenido:Permite estructurar la informacion en niveles (titulos, secciones, parrafos).
  • Compatibilidad universal:Es interpretado por todos los navegadores web modernos.


Estructura basica de un documento HTML:

< !DOCTYPE html>
< html lang="es">
< head>
    < meta charset="UTF-8">
    < meta name="viewport" content="width=device-width, initial-scale=1.0">
    < title>Mi pagina web< /tittle>
< /head>
< body>
    < header>
        < h1>Bienvenido a mi sitio< /h1>
    < /header>
    < main>
        < p>Este es un parrafo de ejemplo.< /p>
    < /main>
    < footer>
        < p>© 2026 Mi pagina< /p>
    
< /body>
< /html>

CSS (Cascading Style Sheets)

CSS es un lenguaje de estilos que se utiliza para describir la presentacion visual de una pagina web creada con HTML. Permite controlar aspectos como colores, tipografias, tamanos, margenes y disposicion de los elementos. Gracias a CSS, es posible separar el contenido de su diseno, facilitando la personalizacion y el mantenimiento. Tambien permite crear disenos responsivos adaptados a distintos dispositivos.

Caracteristicas Clave de CSS:

  • Separacion de diseno y contenido: Permite aplicar estilos sin modificar la estructura HTML.
  • Diseno responsivo: Facilita adaptar la apariencia a diferentes tamanos de pantalla.
  • Cascada y herencia: Los estilos se aplican segun prioridad y pueden heredarse entre elementos.
  • Personalizacion visual: Controla colores, tipografias, espaciados y distribucion de elementos.
